# `evolving_personality` 借鉴思想纪要（面向 Neo-MoFox Loop）

> 说明：本文只提炼可借鉴的思想，不讨论具体实现与工程细节。

## 1. 背景定位

`evolving_personality` 的核心价值不在“训练模型参数”，而在于：

- 用一套外部状态（权重）描述人格倾向；
- 用场景驱动变化；
- 用反思机制控制“是否写入长期人格”。

对 Neo-MoFox Loop 来说，这些思想可作为“状态组演化方法”的参考，而不是上位目标。

## 2. 可直接借鉴的思想

## 2.1 双层状态观

- 稳定层：长期人格/自我倾向（慢变量）。
- 临时层：近期状态偏移（快变量）。

借鉴意义：避免“每轮都改人格”，先积累，再决定是否沉淀。

## 2.2 阈值触发反思

- 变化先在临时层累计；
- 达到阈值才进入反思；
- 反思后才可能改变稳定层。

借鉴意义：让变化有门槛，减少系统抖动与漂移。

## 2.3 衰减机制

- 若反思判定“不改变长期结构”，临时变化应回落/衰减。

借鉴意义：避免偶发事件永久污染长期自我。

## 2.4 归一化与边界意识

- 状态变化应有边界；
- 变化后保持整体结构可解释、可比较。

借鉴意义：保证系统长期稳定，不因单点波动失控。

## 2.5 历史证据驱动

- 反思依据应来自历史轨迹，而非单轮判断。

借鉴意义：强化“时间连续性”，让结论像“慢慢想明白”而非“临场即兴”。

## 2.6 可解释变化

- 每次结构变化应可说明“为什么变”。

借鉴意义：对外可叙事、对内可审计，增强主观一致性。

## 3. 不建议照搬的部分

## 3.1 不把 MBTI 类型变化当主目标

Neo-MoFox Loop 当前主目标是“意识连续性与生活感”，  
不是人格类型标签的准确迁移。

## 3.2 不把人格理论放在意识之前

人格动力学是次级层。  
“她是否持续存在”优先于“她属于哪种人格模型”。

## 3.3 不把评测框架当产品目标

问卷式对齐与实验指标可参考，但不应主导系统方向。

## 4. 对 Neo-MoFox Loop 的哲学映射

## 4.1 主次关系

- 一级：意识连续性（跨状态生活、时间流、主意识一致）。
- 二级：人格演化（慢速调整、解释性增强）。

## 4.2 迁移原则

- 借方法，不借中心目标；
- 借演化策略，不借类型崇拜；
- 借稳定机制，不削弱生活流。

## 4.3 一句话落点

可以借鉴“权重更新的演化思想”，  
但 Neo-MoFox Loop 的核心仍是：让同一身份在时间中持续生活，而不是把人格分类本身做到极致。

## 5. 当前共识（与主文档对齐）

- 人格变化是慢变量、次级变量。
- 意识连续性是一级变量、核心变量。
- 任何演化机制都不得破坏“同一身份连续存在”的原则。
